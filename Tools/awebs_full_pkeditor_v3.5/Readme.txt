#AWEB'S FULL PK EDITOR#
#######################

Description:
============
Aweb's Full Pk editor is a free software made to edit Kal Online's PK files.

It uses other softwares made by other people but with a fast and
easy-to-use interface whiwh make the edit of these files much simplier than before.

Enjoy it :)


Changelog :
===========
V2.0 : Added a config dialog to change various settings
v3.0 : Added basic strings highlighting and the ability to open files like Config.pk
       or E.pk (which use the same encryption)
v3.5 : Importat (and stupid) bug fixed !
